# HTML表单标签

表单：用于采集用户输入的数据，用于和服务器进行交互

​	form：用于定义表单。可以定义一个范围，范围代表采集用户数据的范围

​		属性：

​				action:指定提交数据的URL

​				method：指定提交方式

​						分类一共七种，2种比较常用：

​								***get ：***

​										***1.请求参数会在地址栏中显示。会封装到请求行中（HTTP协议后讲解）***

​										***2.请求参数大小是有限制的。***

​										***3.不太安全***

​								***post :***

​										***1.请求参数不会在地址栏中显示。会封装到请求体中（HTTP协议后讲解）***

​										***2.请求参数大小没有限制。***

​										***3.较为安全***

​			***表单项中的数据要想被提交：必须指定其name属性**
